💥주의사항💥
-------------
</br>
일관성을 위해 커밋메시지는 아래와 같이 맞춰주세요.
</br>

- create : 새로운 기능을 생성
    
    ```
    create : 게시판 페이지 생성
    ```
    
- update : 기존 기능을 리팩토링 또는 수정
    
    ```
    update : 게시판 디자인 변경
    ```
    
- fix : 버그 등 결함등을 해결
    ```
    fix : 결함 해결 내용
    ```

</br>
혹시 PUSH, COMMIT, PULL 등의 절차나 사용법을 모르시거나 헷갈리시다면, 저(정충일)에게 문의해주세요.
